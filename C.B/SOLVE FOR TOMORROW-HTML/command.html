麻烦问一下，提交的文件是啥格式的？
创新科技 西宁赛区

创意作品单元和科普实验单元提交文件的要求各不一样。 创意作品单元需要PDF格式的方案、jpg格式的承诺书拍照或者扫描件、zip或rar格式的设计文件压缩包、MP4格式的视频（非必须）。 科普实验单元需拍摄MP4格式比赛视频、jpg格式物料清单的拍照。 详细提交文件的要求可参见赛事介绍文件。 期待你的参与！^_^
